# 🎼 ScoreForge — 쉬운 웹 악보 에디터

MuseScore의 핵심 기능(음표 입력 → 편집 → 재생 → 저장/내보내기)을 브라우저에서 바로 쓸 수 있게 만든
**설치·빌드 없는 정적 웹앱**입니다. `musescore-web-vibecoding-guide.md`의 아키텍처 원칙
(유리수 시간 모델, 레이아웃/렌더 분리, 단일 편집 통로, MuseScore 단축키 관례)을 따릅니다.

## 실행 방법

| 방법 | 하는 법 |
|---|---|
| 그냥 열기 | `index.html` 더블클릭 (인터넷 연결 시 Bravura 악보 폰트 자동 로드) |
| 로컬 서버 | 폴더에서 `python -m http.server 8000` → http://localhost:8000 |
| 배포 | 폴더 전체를 Render/Netlify/GitHub Pages에 **Static Site**로 올리면 끝 |

> 오프라인에서도 고품질 글리프를 쓰려면 [Bravura.woff2](https://github.com/steinbergmedia/bravura/raw/master/redist/woff/Bravura.woff2)를
> 내려받아 `fonts/Bravura.woff2`에 넣으세요. 없으면 내장 벡터 글리프로 자동 폴백됩니다.

## 주요 기능

- **음표 입력 3가지 방법**: 보표 클릭(고스트 미리보기) · 키보드 A~G · 화면 피아노 건반
- **편집**: 선택/방향키 탐색, **범위 선택(Shift+클릭·Shift+←→)**, 드래그로 음높이, 음길이 변경, 점음표, 임시표(♯♭♮), 화음(Shift), 타이, **잇단음표(Ctrl+3~9)**, 가사, 삭제
- **표현 기호 (가이드 Phase 4)**: 이음줄(슬러, 시스템 경계 분할·음표 회피), 셈여림 pp~ff, 크레셴도/디미누엔도 쐐기, 아티큘레이션(스타카토·테누토·악센트·마르카토, 줄→칸 밀기·세로 쌓기), 페르마타 — 전부 재생(velocity/gateTime/레가토/선형 보간)에 반영
- **조판**: 빔 자동 묶음(박 단위·4/4 8분음표 2박 병합), 마디 경계 자동 타이 분할, 임시표 자동 표기 규칙, 자동 줄바꿈/양쪽 정렬
- **재생**: Web Audio 신디사이저(피아노·일렉피아노·뮤직박스·오르간·현악기·플루트·8비트), 재생 커서·음표/건반 하이라이트, 메트로놈, 선택 위치부터 재생, 템포 조절
- **악보 설정**: 음자리표(높은/낮은), 조표 15종, 박자(2/4·3/4·4/4·6/8, 내용 유지 재배치), 조옮김(반음~옥타브)
- **파일**: 저장/열기(.json), **MusicXML 내보내기·가져오기**(.musicxml/.xml/**.mxl 압축 포함**, MuseScore와 양방향 호환), **MIDI 내보내기**, 인쇄/PDF, 자동 저장(브라우저), **드래그&드롭으로 열기**
- **가져오기 리포트**: 다중 파트/성부·꾸밈음·도돌이표 등 지원 범위를 벗어난 요소는 안전하게 줄이고, 무엇이 바뀌었는지 요약 다이얼로그로 보여줌 (못갖춘마디는 앞쉼표로 자동 정렬)
- **실행 취소/다시 실행** 무제한(200단계), 한국어 UI, 반응형(모바일~데스크톱), 시작 가이드·데모 악보 4곡

## 단축키 (MuseScore 관례)

| 키 | 동작 | 키 | 동작 |
|---|---|---|---|
| `N` | 음표 입력 모드 | `3`~`7` | 16분~온음표 |
| `A`~`G` | 음 입력 | `0` | 쉼표 |
| `.` | 점음표 | `Shift+A~G` | 화음 쌓기 |
| `↑` `↓` | 반음 (Ctrl=옥타브) | `←` `→` | 이전/다음 선택 |
| `T` | 타이 | `L` | 가사 (더블클릭도 됨) |
| `Ctrl+3`~`Ctrl+9` | 선택 음표를 잇단음표로 분할 |  |  |
| `S` | 이음줄(슬러) | `Shift+클릭`/`Shift+←→` | 범위 선택 |
| `Shift+S/N/V/O` | 스타카토/테누토/악센트/마르카토 | `<` `>` | 크레셴도/디미누엔도 |
| `Delete` | 삭제(쉼표로) | `Esc` | 입력 종료/정지 |
| `Space` | 재생/일시정지 | `Ctrl+Z` / `Ctrl+Shift+Z` | 실행 취소/재실행 |
| `Ctrl+S` | 저장 | `Ctrl+휠` | 확대/축소 |

## 코드 구조 (가이드의 레이어 규칙)

```
index.html        앱 셸(툴바·다이얼로그·피아노)
css/style.css     디자인 토큰(다크 작업실 + 흰 종이), 반응형, 인쇄
js/core.js        ★ 도메인: Fraction 시간, tpc식 음높이 철자, 악보 트리,
                    fillGap 편집 연산, rebar, 조옮김, 스냅샷 undo/redo
js/engrave.js     ★ 조판: SMuFL(Bravura) 글리프 + 폴백 패스, 레이아웃(간격·줄바꿈·justify),
                    빔/타이/임시표/덧줄 렌더, 히트 테스트, 고스트/커서 오버레이
js/playback.js    ★ 재생: 악보→이벤트 컴파일(타이 병합), 신스 프리셋, 룩어헤드 스케줄러,
                    메트로놈, MIDI(SMF) 내보내기
js/io.js          저장/열기(JSON), MusicXML 4.0 내보내기, 자동 저장, 데모 악보 DSL
js/app.js         UI 배선: 단축키, 마우스/드래그, 피아노, 재생 커서, 다이얼로그, 토스트
```

의존 방향은 `app → (engrave, playback, io) → core` 한 방향입니다.
악보를 바꾸는 모든 동작은 `core.mutate()` 한 통로만 지나가므로 undo가 항상 안전합니다.
